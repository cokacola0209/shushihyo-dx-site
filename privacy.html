プライバシーポリシー

最終改定日: 2026年6月18日

施行日: 2026年6月18日

本ポリシーは、アプリ「収支表DX」（以下「本アプリ」）における利用者の情報の取り扱いについて定めるものです。

1. 運営者情報

  • 運営者名: Kourabo

2. 連絡先

  • お問い合わせメール: shushihyo.dx@gmail.com

3. 取得する情報

本アプリは、利用者が画面で入力したデータ（実践記録、投資・回収、実消化仕事量、見込み期待値、店舗名・機種名、メモ、タグ、パートナー情報、支出・収入など）を取得します。これらは利用者の端末内にのみ保存され、運営者のサーバーへ送信されることはありません。

なお、本アプリの利用にあたり、アカウント登録・ログインは必要ありません。運営者が利用者の氏名・メールアドレス等の個人を特定する情報を取得・保管することはありません（第2条の連絡先へ利用者ご自身がお問い合わせいただく場合を除きます）。

加えて、広告配信および課金処理のために、以下の第三者サービスが情報を取得・処理する場合があります（詳細は第5〜7条）。

4. 端末内にローカル保存されるデータ

本アプリの記録データは、以下の仕組みで端末内にのみ保存されます。

  • ローカルデータベース（Drift / SQLite）: 実践記録・支出・収入・パートナー（プロフィール）情報など
  • 端末内設定ストレージ（SharedPreferences）: 各種設定値、課金状態のキャッシュ、マスタ（店舗・機種・換金率など）

これらのデータは外部送信されず、端末内に留まります。

5. AdMob による広告関連データ

無料版では、Google AdMob（Google LLC 提供）による広告を画面下部のバナーとして表示します。AdMob は広告配信のために、広告識別子（IDFA / AAID）、デバイス情報、おおよその位置情報（IP アドレスに基づく）、広告の表示・操作に関する情報などを取得・利用する場合があります。

  • 取得・利用の主体は Google であり、その取り扱いは Google のポリシーに従います。
  • iOS では、トラッキング（広告識別子の利用）に先立ち、App Tracking Transparency（ATT）による許可を求めます。許可しない場合でも本アプリの基本機能は利用でき、広告は非パーソナライズで表示されます。
  • 有料版（月額サブスクリプション）では広告は表示されません。
  • Google のプライバシーポリシー: https://policies.google.com/privacy
  • 広告に関する設定: https://adssettings.google.com/

6. RevenueCat による課金関連データ

アプリ内課金（月額自動更新サブスクリプション）の管理に RevenueCat（RevenueCat, Inc. 提供）を利用します。RevenueCat は、購入トランザクション、購入・更新・復元の状態、匿名のアプリ利用者ID（App User ID）などを処理します。

  • クレジットカード番号などの決済情報を、本アプリおよび RevenueCat が取得することはありません（第7条のとおり Apple / Google が処理します）。
  • RevenueCat のプライバシーポリシー: https://www.revenuecat.com/privacy

7. App Store / Google Play による決済処理

サブスクリプション課金の決済および定期更新は、Apple（App Store）または Google（Google Play）を通じて行われます。決済手段やカード情報などの決済関連情報は、各ストア（Apple / Google）が取得・処理します。

  • Apple のプライバシーポリシー: https://www.apple.com/legal/privacy/
  • Google のプライバシーポリシー: https://policies.google.com/privacy

8. データの書き出し・共有機能の扱い（CSV / JSON バックアップ）

本アプリには、記録データを利用者自身が書き出す・共有する機能があります。

  • CSV 出力 / JSON バックアップの作成: 端末内のデータをファイルとして書き出します。
  • 共有: 書き出したファイルを、利用者が選んだ送信先（メール、クラウド、他アプリなど）へ共有できます。
  • ファイル選択・復元: 端末や外部ストレージからバックアップファイルを選び、復元できます。

これらは利用者が明示的に操作した場合にのみ実行され、送信先の選択も利用者に委ねられます。共有・保存先での取り扱いは各サービスのポリシーに従います。

9. 利用する第三者 SDK

  • Google AdMob（広告配信） — Google LLC
  • RevenueCat（アプリ内課金管理） — RevenueCat, Inc.

10. データの保存場所

本アプリの記録データは、原則として利用者の端末内に保存されます。広告・課金に関するデータは、各第三者サービス（Google / RevenueCat / Apple）の管理下で処理されます。

11. データの削除方法

  • アプリ内の初期化／データ削除機能を利用する。
  • 本アプリをアンインストールする（端末内に保存された記録データは削除されます）。
  • 課金・広告に関して第三者が保持するデータの削除・設定変更については、各社のポリシー・設定をご利用ください。

12. お子様の利用について

本アプリはお子様（子ども）向けに設計・提供されているものではありません。

  • 表示される収支・期待値・仕事量は、利用者の入力に基づく記録であり、勝敗や利益を保証するものではありません。
  • 遊技は20歳以上を対象とし、各自の責任において節度を持ってご利用ください。

本ポリシーは、必要に応じて改定されることがあります。重要な変更がある場合は、アプリ内またはストア上で告知します。改定後の本ポリシーは、本アプリ上に表示された時点から効力を生じます。
